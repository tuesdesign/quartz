# About

This is a game which is not designed to be fun. You should not climb the tower. There is nothing of any value waiting at the top. You should not climb the tower. This game is difficult, patently unrewarding, and intentionally frustrating. You should not climb the tower. The tower is fair. Your mistakes are yours alone. You should not climb the tower. The tower is not a metaphor for human will, or a trick to see what people will endure. You will not learn anything. You will not gain anything. 

You should not climb the tower, but you _can_.

# Portfolio Note
This game was created in 24 hours as part of a game jam. My goal was to create a game that is immensely frustrating but fair. To this day, I think I'm the only one who's seen the 'ending'.
